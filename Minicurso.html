🐍 Mini Curso de Python

Aprende Python desde cero de forma fácil y rápida

¿Qué es Python?

Python es uno de los lenguajes de programación más populares del mundo. Se utiliza para crear páginas web, inteligencia artificial, videojuegos, automatización y análisis de datos.

Primer programa

La función print() sirve para mostrar texto en pantalla.

print("Hola Mundo")

Variables

Las variables almacenan información.

nombre = "Axel"
edad = 16

print(nombre)
print(edad)

Tipos de datos

  • String → Texto
  • Integer → Números enteros
  • Float → Decimales
  • Boolean → Verdadero o falso
texto = "Python"
numero = 10
decimal = 3.14
activo = True

Entrada de datos

Con input() el usuario puede escribir información.

nombre = input("Escribe tu nombre: ")

print("Hola", nombre)

Operadores Matemáticos

suma = 5 + 3
resta = 5 - 3
multiplicacion = 5 * 3
division = 5 / 3

print(suma)

Condicionales

Permiten tomar decisiones.

edad = 18

if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")
